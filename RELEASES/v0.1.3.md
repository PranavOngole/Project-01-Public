## Context

The original conviction surface returned a single intrinsic-value scalar for every name. That number papered over a real uncertainty: the same DCF inputs yield a range, not a point, and the width of the range is itself a signal. Earlier internal review found this implicitly required users to act on a precision the engine did not actually have.

This release replaces the scalar with a distributional view and adds an empirical calibration test that must pass before users see the new representation.

## Outcomes

**Intrinsic value is now displayed as a quantile band.** P5 through P95 with a labelled dominant-uncertainty driver. The band itself does the work the prior single number did, but readers can see the spread.

**Dual-layer capital-exposure discipline.** The recommendation layer continues to enforce a fundamental margin-of-safety threshold. A trade-execution layer adds a second numeric check at fifteen percent below the band median. The two layers are independent; either can block a position.

**Position sizing follows a four-tier conviction ladder.** Zero size above the band median. One-third allocation in the upper-half tail (median to lower quartile). Two-thirds allocation in the lower-half tail (lower quartile to lower extreme). Full size below the lower extreme. Sizing is mechanical rather than discretionary.

**Release gate redefined after empirical evidence.** The originally specified gate proved structurally unreachable for any name trading materially away from DCF fair value. The replacement gate measures whether the band's width matches the asset's empirical short-horizon return spread rather than whether realized returns land inside the band's implied-return zone. The replacement passes on the test cohort at a ratio of 0.795x against an empirical reference of 1.0x within the accepted band.

**Methodology limits disclosed.** The current release validates band width, not band centering. Centering validation requires point-in-time fundamentals across multiple regimes and remains queued for a later release.
