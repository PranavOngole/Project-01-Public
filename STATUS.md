# Status

A plain, no-spin account of where Project-01 actually is. The README is the
pitch. This file is the reality check. Read both.

## Stage

Early. Pre-1.0, under active development. The current release is **v0.1.4**
(2026-05-14). See [`CHANGELOG.md`](./CHANGELOG.md) and [`RELEASES/`](./RELEASES/)
for the exact scope of every version.

## What is true

- **Built by one person.** This is a solo project, not a funded team or a company.
- **Execution is paper-only.** No real capital is at risk. The broker integration
  is pinned to paper trading by design.
- **One external review.** An external code review (CR-01) was commissioned, and
  its findings were remediated in v0.1.2. That is a single review, not a
  certification or an ongoing audit.
- **Coverage is narrow.** Calibration evidence spans five tickers as of v0.1.4.
  The system has not been validated across the broad market.

## What this is not

- Not financial advice, and not a recommendation to buy or sell any security.
- Not a production system for managing real money.
- Not autonomous trading. Nothing executes without explicit human authorization.
- Not a signals service, and not a guaranteed-returns product.
